=>  In bootstrap there is a class offset it is use to create space between col's inside row 

=> the range of this offset is offset-1 to offset-11


=>  you can also use six different bootstrap breakpoints in offset class to adjust       different space between the col in different screen size  

Breakpoints points                              Classes

(1) Extra small: <576px                         None    
(2) small: 	≥576px to 768px                     sm
(3) Medium: ≥768px to 992px                     md
(4) Large: 	≥992px to 1200px                    lg
(5) Extra large: ≥1200px to 1400px              xl
(6) Extra extra large: ≥1400px                  xxl


New In Bootstrap 5 

(6) Extra extra large: ≥1400px      xxl
